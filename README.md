 SCRABBLE
 ===

# About

Scrabble with replicas

# Requisites

- Ubuntu
- Python (& pip)
- Word list         `sudo apt install wbrazilian`
- Tkinter           `sudo apt install python-tk`
- Sorted Containers `sudo pip install sortedcontainers`
